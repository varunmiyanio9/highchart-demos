Selected Points Detail

Click, Ctrl+Click, or drag to select bars. Use Tab → Arrow keys → Space for keyboard selection. Shift+Arrow for range select.

Bucket Selection — Multi Chart Types

Selection is shared across all charts. Click, Ctrl+Click, or drag to select buckets.

Selected Buckets Detail

Click to select entire category. Ctrl+Click to multi-select. Drag to select range. Keyboard: Tab into chart → Arrow keys → Space to toggle. Escape to clear.

Selected Categories Detail

Click buttons to select categories. Ctrl+Click for multi-select. Shift+Click for range. Keyboard: Tab into buttons → Arrow keys to move → Space to toggle. Escape to clear.

Selected Categories Detail

Click x-axis labels to select. Ctrl+Click for multi-select. Click chart background to clear.

Combined Selection (SVG) — Bidirectional Label + Grid

Drag on chart background for range select. Click x-axis labels. Ctrl+Click for multi-select. Keyboard: Tab → Arrow → Space.

Selected Categories Detail

Drag to select range on grid. Click labels for single select. Ctrl+Click for multi. Shift+Arrow for range. Escape to clear.

Forecast Charts — Historical vs Predicted

Each chart shows historical data (solid) and forecast data (dotted/translucent) with confidence bands. Orange line marks the forecast start.

Context Menu — Custom Right-Click

Final Version — Combined Chart

The finalized presentation: combined chart types, three stacked groups with one grouped legend, category selection (mouse + keyboard), right-click type switching, a forecast divider behind the bars, and a horizontal scroll toggle.

Selected Months Detail

Final Version HBC — Horizontal Bar Charts

Two independent horizontal-bar demos. Toggle Allow Scroll to scroll the bars vertically (the value axis stays sticky). Change font size / weight. Select categories: drag on the chart background, click an axis label (Ctrl/Cmd+Click to multi-select), or use the keyboard (Tab → Arrow → Space/Enter, Shift+hover for a range, Escape to clear). The single Enable Point Label checkbox turns values on for both charts.
